# Protocol Services

Protocol services built on the transport layer providing protocol-specific functionality.

## Architecture Overview

Protocol services integrate with transports to handle protocol-specific logic:

```
┌─────────────────────────────────────────┐
│ Protocol Service                        │  ← Protocol-specific implementation
├─────────────────────────────────────────┤
│ Middleware                              │  ← Service-level middleware
├─────────────────────────────────────────┤
│ Transport Layer                         │  ← HTTP / WebSocket
├─────────────────────────────────────────┤
│ Transport Middleware                    │  ← Transport-level middleware
├─────────────────────────────────────────┤
│ Network                                 │  ← Network communication
└─────────────────────────────────────────┘
```

**Request Flow:** Protocol Service → Middleware → Transport → Transport Middleware → Network

**Response Flow:** Network → Transport Middleware → Transport → Middleware → Protocol Service
